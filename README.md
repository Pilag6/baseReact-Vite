# baseReact with Vite

## Install & Create New Folder
```
    npm create vite@latest <folderName> -- --template react 
```

## Install in a current folder already created
```
    npm create vite@latest . -- --template react 
```

```
    cd <folder-name>
    npm install
    npm run dev
```
